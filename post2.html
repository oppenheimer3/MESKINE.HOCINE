Stocks prediction GUI using API

Summary

The goal of this project was to predict the stocks of certain company based on the previous stocks using an API, then pack it into a graphical user interface.

The data

The data of this project was extracted from the stock API Polygon.io â€“ Stock Market Data APIs, then turned into a pandas dataframe:


The open price (or closed) stocks column was transformed into values in the range of [0,1] using the MinMaxScaler and then was divided into train set 80% and test set 20%, the train data was finally made by taking the first 60 values as X and the 61th value as the corresponding Y and so on.

The machine learning model

Long Short Term Memory was used in this project which is a type of recurrent neural network capable of learning order dependence in sequence prediction problems. So after building the network and fitting the model on the train data, the model was used on the test data that was built in the same way as the train data to predict the stocks:
